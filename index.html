Homework #4 Селекторы, Специфика

Пример селектора по типу элемента (p). Селектор p выберет все элементы абзаца (p) на странице и применит указанные стили к этим элементам.

Пример селектора по id (#element-with-id). Селектор с id, такой как #element-with-id, выберет только элемент с уникальным идентификатором "element-with-id".

Пример селектора по классу (.element-with-class). Селектор по классу, например, .element-with-class, выберет все элементы, которые имеют указанный класс "element-with-class".

Пример селектора по нескольким классам (.element-with-class и .multiple-classes). Селектор выберет только элемент, который имеет оба класса.

Пример селектора по вложенности (div p). Селектор div p выберет все элементы абзаца (p), которые находятся внутри элементов div на странице.
