  • git clone url-адрес - Скачивает репозиторий вместе со всей его историей изменений
  • git status - Разрешает просматривать список измененных файлов
  • git add - Добавление изменений для будущего коммита
  • git commit -m "сообщение с описанием" - Для создания коммита в Git с сообщением коммита
  • git log - Просмотр и изучение истории изменений файлов проекта
  • git push - Отправка локальных изменений в репозиторий
  • git checkout -b name - Создание новой ветке и сразу переключение на нее
  • git commit -am "text" - Создания коммита сразу с изменениями в файле
  • git branch - Разрешает посмотреть все ветки и ветку на которой мы находимся
